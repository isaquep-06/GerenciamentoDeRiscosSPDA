Fluxograma do Protótipo de Cálculos (SPDA)

Visual de alto nível do fluxo que você vai programar: Inputs → Cálculos → Comparação → Resultado.

1) Entrada de Dados

NG L W H CD

Usuário informa os valores no formulário.

2) Cálculos

AD = (L×W) + 2×(3×H)×(L+W) + π×(3×H)²

ND = NG × AD × CD × 1e-6

ADÁrea exposta
NDDescargas estimadas
RTLimite
ND vs RTComparação

3) Resultado

Se ND > RT → ⚠️ Necessita proteção

Se ND ≤ RT → ✅ Estrutura protegida

RT (vida humana) padrão usado no protótipo: 0.00001
